Notepad ++

What is Notepad ++

Notepad++ is a free source code editor and Notepad replacement that supports several languages. It drastically overhauls the default notepad app and added a multitude of new futures. Like the ability to have multiple tabs open in one menu, many different color themes, and many more. Find Out More. Notepad++ also allows the use of plugins so you can either download or make your own to add any functionality you need.
