Notepad ++

What is Notepad ++

Notepad++ is a free text editor and Notepad replacement that supports several languages. So basically it takes the default windows notepad program and completely overhauls it. Unlike the regular notepad where you can only open a limited number of file types and when it does open them it most of the time displays them as a jumbled mess. Notepad++ opens many different files types and displays them in a nice and customizable way. Notepad ++ comes with many help features, like the ability to have multiple tabs open in one menu, the ability to make record and save custom macros, and many more (Find Out More). On top of these features you also have the ability to install plugins so if there is missing feature or language you want you can most likely find a plugin that adds it in.

Installation

Step 1

Go to the notepad++ website (https://notepad-plus-plus.org/)and go to the download page. Once your there look under Download 64bit and click "Notepad++ Installer 64-bit" to download the 64bit version. If you just want the 32-bit version you can just click the big green download button.

Step 2

Now that you have the installer downloaded double click to run it and select your language. You can then just click next and agree to there term of service.

⤷

Step 3

Next your going to want to choose where you want to install Notepad++ at. By default it install into your programs folder on your C: drive, but you can choose to install it where eveer you want.

⤶

Step 4

Now you can decide what components you want to install along with it. By default it it selects all of them, which I also suggest because they don't take up much space, but you can choose what you want.

⤷

You're done!

And with that you have successfully installed notepad++ and should have something similar to this.

Overview

Now that you have it installed you can go to Setting-> Style Configurator and choose a different color theme if you wish. You can also create a custom theme and make it look however you want. I also suggest going to Setting-> Preferences -> auto-complete and turn on any of the auto-complete. Another place to check out in setting is the short cut mapper. this is where you can see and change around all of the keyboard commands that can be done and edit any macros you make.

Along the top they give you controls to toggle world warp, which just makes it so text stays on screen instead of running off to infinity, show all characters, define your own language, see a map of your document, see what function you have in a program ,set your folder as a workspace, and record, save,and run custom macros.
